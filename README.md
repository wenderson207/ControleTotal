# 📊 Controle Total — Pequenas Lojas

Landing page de vendas para a **Planilha Controle Total**, desenvolvida para pequenas lojas como farmácias, eletrônicos, roupas e mercadinhos.

## 🚀 Sobre o Projeto

Uma página de vendas moderna e otimizada para conversão, com:
- Design dark premium responsivo
- Seções de dor, solução, comparação e depoimentos
- FAQ interativo com animações
- Contadores animados
- Botão sticky de CTA
- 100% HTML/CSS/JS — sem dependências externas

## 📁 Estrutura

```
ControleTotal/
├── index.html        # Landing page principal
├── assets/           # Imagens e recursos adicionais
└── README.md         # Este arquivo
```

## 🛠️ Como usar

Basta abrir o `index.html` em qualquer navegador, ou hospedar em:
- GitHub Pages
- Netlify
- Vercel
- Qualquer servidor web

## 🎯 Segmentos atendidos

- 🧴 Farmácias
- 📱 Eletrônicos
- 👗 Roupas e Moda
- 🛒 Mercadinhos
- E qualquer pequena loja!

## ⚙️ Personalização

Para personalizar a página:
1. Abra `index.html` em um editor de texto
2. Substitua os textos de placeholder pelo nome da sua marca
3. Atualize o link de compra no botão `COMPRAR AGORA`
4. Adicione seu CNPJ no rodapé
